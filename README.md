# sales-insight-dashboard
Automated Sales Dashboard analyzing over 1,000+ rows of raw transaction data. Built with Power Query for ETL, Pivot Tables for dynamic KPI tracking, and interactive Slicers for granular data exploration.

**Key Features**
* Automated ETL: Processed raw data using **Power Query**(Cleaning, Transforming, and Loading).
* Dynamic KPIs: Real-time tracking of Total Sales, Profit Margins, and Transaction Volumes.
* Interactive Exploration: Intergrated Slicers and Timelines for granular filtering by Year, Region and product category.
* Visual Storytelling: Used advanced charting tecniques to highlight sales trends and underperforming markets.

**Tech stacks**
* Analysis and visualization: Microsoft Excel/Power BI
* Data processing: Power Query
* Components: Pivot Tables, Slicers, and dynamic charts
